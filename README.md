# SpringBoot-demo1
搭建一个 SpringBoot + Hibernate + Gradle 项目

一、Srping Boot
+ spring boot 出现就是为了替代传统的配置方式，使用了注解的方式和 java 进行配置。 
+ 演变：xml、Annotation、Java Config
+ 特性：
  - Starter （功能依赖）
  - AutoConfiguration （自动配置）
  - Cli （操作）
  - 监控 （如：catuator）
